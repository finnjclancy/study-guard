# 🛡 Study Guard

A Chrome extension that blocks distracting websites by redirecting them to a destination of your choice.

## Features

- ✅ Whitelist mode — everything is blocked except sites you explicitly allow
- ⚡ Instant redirect option — skip the block page entirely
- 🔊 Optional max-volume alarm as a deterrent
- ⚙️ Fully customizable whitelist and redirect URL
- 🔒 100% local — no data leaves your browser
